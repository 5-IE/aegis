# Aegis Project
An IoT Project to automatically record attendance of students by entering the location of the building. 

## Challenge Response: 
Create an app that utilize CoreLocation to let User (Learners) to just walk pass by the detection system and get recorded whether they clocked in or clocked out to the User (Admin) device

## Use Cases
1. User can register and generate new id in their device to the detection system
2. User can just walk pass by the detection system and get notified whether they clocked in or clocked out in their device
3. User and Admin can get a daily absence recap

  
## Tech Stack:
  
1. IoT
Directory: Aegis-Beacon
Device: ESP32
  
2. Frontend for User
Directory: Aegis
Language: SwiftUI
OS: iOS App
  
3. Frontend for Admin
Directory: Aegis-Admin
Language: SwiftUI
OS: MacOS App
  
4. Backend
Directory: Aegis-Backend
Language: NodeJS, Typescript (Express)
